# 🎟️ Movie Ticket Booking Chatbot

## 📌 Project Title

**Movie Ticket Booking Chatbot using Python (Tkinter)**

---

## 🎯 Aim

To design and develop a **GUI-based chatbot** that allows users to book movie tickets through a conversational interface.

---

## 📝 Description

This project is a **desktop chatbot application** built using Python and Tkinter.

It simulates a ticket booking system where users can:

* Select a movie
* Choose seats
* Select payment method
* Confirm booking

The chatbot guides users step-by-step like a real booking assistant.

---

## 🎯 Objectives

* Automate ticket booking process
* Provide interactive user experience
* Demonstrate chatbot workflow using Python
* Implement GUI-based chatbot

---

## ⚙️ Working Principle

The chatbot works using a **state-based flow system**.

### Flow:

1. User types "book"
2. Bot asks for movie
3. User selects movie
4. Bot asks for seat
5. User selects seat
6. Bot asks for payment
7. Booking confirmed

---

## 🧠 Algorithm

1. Initialize chatbot
2. Set stage = start
3. Accept user input
4. Check current stage:

   * Start → Ask for booking
   * Movie → Select movie
   * Seat → Select seat
   * Payment → Process payment
5. Move to next stage
6. Display response
7. Repeat
